# Odin Calculator

A browser-based calculator built as part of [The Odin Project](https://www.theodinproject.com/) Foundations curriculum.

## Features

- Basic arithmetic operations: addition, subtraction, multiplication, and division
- Decimal point support
- Backspace button to delete the last entered character
- Clear button to reset the calculator
- Division by zero protection
- Dynamically generated buttons from a configuration object

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript

## What I Practiced

- DOM manipulation and dynamic element creation
- Event listeners with callbacks
- Lookup tables as an alternative to `switch` statements
- DRY principles and function abstraction
- `Object.entries()` and `for...of` with destructuring
- Edge case handling (division by zero, decimal point duplication, backspace on single digit)
- JSDoc documentation

## How to Use

1. Clone the repository:
   ```bash
   git clone git@github.com:YoshiBlaA/odin-calculator.git
   ```
2. Open `index.html` in your browser.
